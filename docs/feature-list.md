# 機能一覧

## F01. 倫理マップの表示

対応UC: UC001

- トップページにアクセスするとプログラマー倫理マップを表示する
- 同心円状のビジュアルでノード (N0, N1-1〜N1-3, N2-1〜N2-6) を配置する
- Pノードをマップ左側に表示する

## F02. ノード説明の表示

対応UC: UC002

- ノードをマウスオーバーまたはクリックすると、そのノードの説明を表示する
- 選択中のノードを強調表示する

## F03. ポイント配布の開始

対応UC: UC003

- 「配布開始」ボタンを押すと Pノードに10ポイントを付与する
- 配布モードに切り替わり、各ノードへのポイント割り振りが可能になる

## F04. Pノードへのポイント追加

対応UC: UC003

- 配布モード中に「ポイント追加」でPノードのポイントを増やせる

## F05. ノードへのポイント割り振り

対応UC: UC003

- 配布モード中、ノードを選択してポイントを増減できる
- ルール:
  - 上限: 親ノードのポイント以下
  - 総量: Pノードのポイントを消費するため、Pノードが0になるとそれ以上増やせない
  - ポイントを減らすとき、子ノードの現在値がルールに違反する場合は減らせない
- ルール違反の操作をしようとした場合、わかりやすいフィードバックを返す

## F06. ポイントのリセット

対応UC: UC003

- 「リセット」ボタンで全ノード (Pノードを含む) のポイントを0にする

## F07. 自分のマップの保存・共有

対応UC: UC004

- ユーザーが名前を入力できる
- 現在のマップ状態 (ユーザー名 + 全ノードのポイント) を URL に encode する
- URL をクリップボードにコピーできる
- URL を開くと保存されたマップ状態を復元する

## F08. 倫理綱領テキストの生成 (オプション)

対応UC: UC003 (6番)

- ローカルの生成 AI (ollama) が利用可能な場合のみ有効
- ユーザーのポイント配分をもとに、倫理綱領テキストを生成・表示する
- AI が利用不可の場合は非表示にする
