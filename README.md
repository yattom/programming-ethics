# プログラマーの倫理アプリ

本アプリはプログラマーの倫理について触れたり、自分の倫理観について考えたりすることのきっかけになることを目的としています。


## なにができるの？

- 倫理観念をシンプルに整理した「プログラマー倫理マップ」を表示できます
- マップ上のエリアに対応する説明を表示できます
- マップ上のエリアに自分なりにポイントを割り振って、自分自身の倫理マップを動的に作成できます
- ユーザーが作った倫理マップについて生成AIが説明やコメントを加えます
- (TBD)作った倫理マップを使って、倫理的チャレンジをこなすゲームが遊べます


## 技術スタック概観

- Vue.jsで構築される静的フロントエンドアプリです
- バックエンドはなく、利用者のPC上で単体で起動したり、AWSなどに静的Webアプリとしてデプロイできます
